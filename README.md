# Glass

Physics game about 3 Dimensional Fractals and moving a glass marble to a goal

## Building
### MacOS
I'm not going to compile a MacOS binary. Ya'll can figure it out ;)

### Linux
* `cd Glass`
* `mkdir build && cd build`
* `cmake ..`
* `cd ..`
* `cmake --build build`
* `cp build/Glass ./`
